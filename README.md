#Win_sync_script

Легковесное решение для автоматической двусторонней синхронизации директорий, реализованное стандартными средствами Windows (без использования сторонних утилит типа rsync или rclone).

###🚀 Основные возможности

    Bi-directional Sync: Копирование недостающих файлов и обновление устаревших версий в обе стороны.

    Smart Compare: Сравнение файлов по дате последнего изменения и размеру.

    Exclusion List: Гибкая настройка игнорируемых директорий и файлов.

    Automated Tasks: Скрипт оптимизирован для запуска через «Планировщик задач Windows» (Task Scheduler) для работы в фоновом режиме.

###🛠 Технические особенности

    Zero-dependency: Не требует установки дополнительного ПО, работает «из коробки» на любой современной Windows.

    Native Shell: Написано на [PowerShell / Batch — выбери нужное].

###📋 Использование

    Настройте пути к исходной и целевой папкам в начале скрипта.

    (Опционально) Добавьте исключения в массив $exclude.

    Запустите скрипт вручную или добавьте в планировщик.
